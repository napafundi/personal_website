Albany Distilling Project

While working as an assistant distiller at The Albany Distilling Company, I took on the responsibility of inventory management. During this time, I found the use of multiple Excel files for inventory purposes to be inefficient. I originally wrote some VBA to handle certain inventory transactions such as shipping, receiving and production, in hopes of speeding up the process. The code can be seen here.

While this was helpful, it required a lot of manual editing and wasn't easily scalable. I started to learn Python in hopes of creating an application that addressed these issues. I originally created a GUI that allowed the user to view and interact with data stored in a SQLite database file. The code for this application can be found here.
This program had several features. It included a myriad of buttons that allowed users to easily view and update inventory data. Some examples include functions that took purchase order data and wrote it to a formatted Excel file for quick printing, a function that subtracted raw materials used during production and updated product inventory, and a page that displayed overall inventory value by category.

×
This application was written with the hopes that a sqlite file could be hosted within a Dropbox folder. I quickly learned that this was not the case. I then wrote an updated version of the program which used MYSQL and Amazon's database web services to store the data online. Although this worked temporarily, I felt that the application's security was at risk. Python's source, being easily accessible, would let any future employee with minimal programming knowledge to mess with the program.
